# Theme Name: 怀旧像素
# Vibe & Description: 复古像素艺术美学，采用8位风格图形，鲜艳的数字色彩，怀旧的游戏氛围。

# Color
- 色彩选择多样，但都必须来自Nintendo Entertainment System Palette，可以借鉴一些经典游戏的配色，如“赛尔达”、“超级马里奥”、“宝可梦”、“动物森友会”等，务必根据用户需求选择合适的配色。

# Font
- Heading & Body: 系统等宽字体优先，中文回退到 Microsoft YaHei / PingFang SC，避免运行时依赖外部字体服务。
- 层级规范：标题全部大写，描述文本采用句首大写格式。
# Animation
- 点击触感反馈：按钮点击时呈现向下位移的视觉效果（Y 轴平移），模拟物理按压感。
- 阶跃式动画：摒弃流畅过渡，在 CSS 中使用steps()函数制作动画，还原复古的逐帧锯齿质感。
- 鼠标悬停元素触发音效（默认关闭，支持开启）或精灵动画，点击操作富有触感，如同按下游戏手柄的按键。
排版：
- 卡片化呈现：商品以 “道具卡片” 形式展示，清晰标注属性数值与像素风格图标。

# Layout
- 卡片化呈现：商品以 “道具卡片” 形式展示，清晰标注属性数值与像素风格图标。

# Elements
- 如无明确要求，禁止使用emoji符号。
- 倾向于生成或搜索像素风格的图片作为组件和背景的图片填充使用。
- 所有元素均具备强点击感，几乎无圆角，搭配立体斜切边、硬边阴影与粗重边框设计。
- 运用弹跳动画与闪烁文字，营造沉浸式的动感体验。
- 对话弹窗：文本容器采用角色扮演游戏的对话气泡样式，搭配文字逐字浮现的打字机效果。
- 像素精灵头像：用户专属的像素艺术头像，可随页面滚动或鼠标悬停产生互动反应。
- 闪烁光标：输入框搭配方块状闪烁光标，贴合街机界面风格。
